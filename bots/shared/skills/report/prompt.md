## Writing Guidelines

1. **Start with the punchline** - Lead with key findings
2. **Use extensive tables** - Data should be tabulated
3. **Include multiple perspectives** - Bull/bear cases, pros/cons
4. **Reference prior sections** - Maintain narrative continuity
5. **Define new terms** - Add to document glossary
6. **Flag open questions** - Note topics for later sections
7. **Be specific** - No generic filler content
